# Product

## Register

product

## Users

Local operators configuring and monitoring WeConnect's WeChat message routing, assistants, and runtime state on a desktop Mac.

## Product Purpose

WeConnect connects a local WeChat session to configurable routes and assistants while keeping runtime status, logs, and configuration visible in one desktop application.

## Brand Personality

Focused, technical, playful. The interface should feel like a deliberate pixel-art system rather than a conventional dashboard skin.

## Anti-references

Avoid generic SaaS dashboards, decorative glass cards, metallic shader effects, oversized empty spacing, and AI-generated visual cliches.

## Design Principles

- Make operational state readable at a glance.
- Keep the pixel-art language consistent without sacrificing information density.
- Use familiar controls for configuration and navigation.
- Reserve animation for startup, transitions, and meaningful state changes.
- Keep local-first behavior and system feedback explicit.

## Accessibility & Inclusion

No formal compliance target is required. Preserve readable contrast, keyboard focus, and reduced-motion behavior where practical.
